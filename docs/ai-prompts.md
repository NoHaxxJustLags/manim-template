# AI Prompt Templates for Manim

Use these prompts with AI assistants to generate Manim scenes quickly.

> [!TIP]
> The best model at creating videos with Manim is: `Claude Opus 4.5`
> But it's also the most expensive model avialable. Use it wisely and with care.
> For simpler tasks other models like `GPT-5.1-Codex-Max` will do it as well.

## Basic video creation

```text
You are tasked with creating a full learning video about the topic `<Your Topic>` for a <math/physics/CS> class in highschool.
To create this video I want you to use the `Manim` framework and maintain a similar style to 3b1b's videos.
You must at least:
- <Content...>
- ...


For the images please use simple placeholder SVGs. I will replace them with real and fitting images afterwards. 
For any futher questions feel free to ask me.
```
